# Hardware for AI/ML

**Name:** SRI CHANDANA REDDY VANUKURI  
**Course:** ECE 410/510 Spring 2026  
**Project Topic:** Hardware Accelerator for Dense Neural Network Linear Layer (MAC Array Chiplet)





\## Project Description



This project designs a hardware accelerator for a neural network linear layer. The accelerator performs matrix multiplication using multiply-and-accumulate operations, which are common in AI workloads. The goal is to improve performance compared to software by using a custom hardware design connected to a host through AXI4-Lite.
